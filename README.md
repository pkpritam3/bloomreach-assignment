# Run Application

Refer below steps to start application.

## Start Gateway Service

Run following commands to start Gateway service.

```bash
cd gateway
npm install
npm start
```

## Start Web Client

Run following commands to start web client.

```bash
cd web-client
npm install
npm start
```
